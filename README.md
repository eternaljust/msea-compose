## Msea Compose

一款虫部落搜索论坛第三方 App，使用 Jetpack Compose 开发，采用最新的 Material Design 3 设计，支持 Android 10 及以上，适配 Material You 壁纸变化的配色方案，动态颜色适用于 Android 12 及更高版本。
